# 🏛 Клуб Незаменимых — Community Skills

> Пока все учат навыки — мы строим системы, которые их производят.

Это открытый репозиторий **Клуба Незаменимых** — сообщества людей, которые с помощью AI делают в 5 раз больше, не работая в 5 раз больше.

Здесь хранятся **комьюнити-skills** — готовые AI-агенты, созданные участниками клуба. Каждый скилл — это не просто промпт. Это система: инструкция + датасеты + примеры, которую можно взять и сразу применить.

## 🧩 Что такое скилл

Скилл — это упакованный AI-агент, готовый к использованию. Не разовый промпт, а переиспользуемая система.

Каждый скилл содержит:

- **SKILL.md** — системный промпт и инструкция для агента: роль, поведение, ограничения, формат ответа
- **datasets/** — базы знаний, на которые агент опирается (гайдлайны, стандарты, правила)
- **examples/** — примеры входных данных и ожидаемых результатов

Принцип: один скилл — одна конкретная задача, решённая на уровне эксперта.

---

## 🚀 Как использовать

1. Выбери нужный скилл в папке `skills/`
2. Открой `SKILL.md` — там описано, что агент делает и как с ним работать
3. Подключи датасеты из `datasets/` как контекст
4. Запусти агента в своём инструменте (Claude, ChatGPT, Cursor, etc.)

Каждый скилл — самодостаточный. Открыл, прочитал, применил.

## 🤝 Как добавить свой скилл

Мы строим это вместе. Если ты создал агента, который решает конкретную задачу — поделись им с сообществом.

### Требования к скиллу

- Решает **одну конкретную задачу** (код-ревью, генерация контента, анализ данных, и т.д.)
- Содержит `SKILL.md` с чёткой инструкцией
- Включает датасеты, если агент на них опирается
- Протестирован на реальных задачах

### Как контрибьютить

1. Форкни репозиторий
2. Создай папку `skills/<название-скилла>/`
3. Добавь `SKILL.md`, датасеты и примеры
4. Открой Pull Request с описанием: что делает скилл, для кого, какой результат
